# API Specification

Base URL: `http://localhost:8000`  
Interactive docs: `http://localhost:8000/docs` (auto-generated by FastAPI)

---

## Always-Present Endpoints

### Health Check
```
GET /api/health

Response 200:
{
  "status": "ok",
  "version": "0.1.0"
}
```

### Analyze Dataset
```
POST /api/analyze

Request:
{
  "filename": "dataset.csv",       // file must exist in data/raw/
  "analysis_type": "summary"       // "summary" | "detailed"
}

Response 200:
{
  "success": true,
  "results": {
    "format": "csv",
    "row_count": 1000,
    "columns": ["col1", "col2"],
    "statistics": {},              // populated when analysis_type="detailed"
    "sample": [],                  // first 5 rows when analysis_type="detailed"
    "insights": []
  }
}

Response 404:
{ "detail": "Dataset not found in data/raw/: filename.csv" }
```

### Get Results
```
GET /api/results

Response 200:
{
  "timestamp": "2026-06-17T09:00:00",
  "data": {},
  "statistics": {}
}
```

## Frontend Integration

All endpoints return `application/json`. CORS is open for local development.

```typescript
const res = await fetch("http://localhost:8000/api/analyze", {
  method: "POST",
  headers: { "Content-Type": "application/json" },
  body: JSON.stringify({ filename: "dataset.csv", analysis_type: "detailed" }),
});
const { success, results } = await res.json();
```
