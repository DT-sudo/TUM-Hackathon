## Traffic Forecasting Endpoints

### Generate Forecast
```
POST /api/forecast

Request:
{
  "corridor": "A8E",          // "A8E" | "A93S"
  "direction": "outbound",    // "outbound" | "inbound"
  "date_from": "2026-07-01",  // ISO date string
  "date_to": "2026-08-31"     // ISO date string (max 366 days from date_from)
}

Response 200:
{
  "success": true,
  "corridor": "A8E",
  "direction": "outbound",
  "date_from": "2026-07-01",
  "date_to": "2026-08-31",
  "forecast": [
    {
      "date": "2026-07-01",
      "day_of_week": "Wednesday",
      "daily_category": 3,
      "daily_color": "orange",
      "daily_color_hex": "#E67E22",
      "estimated_daily_vehicles": 28000,
      "time_slots": [
        {
          "slot": 1,
          "label": "00:00–06:00",
          "category": 1,
          "color": "green",
          "color_hex": "#2ECC71",
          "estimated_vehicles": 1200
        },
        {
          "slot": 2,
          "label": "06:00–10:00",
          "category": 3,
          "color": "orange",
          "color_hex": "#E67E22",
          "estimated_vehicles": 7500
        }
      ],

Response 422:

Response 500:

```

## Frontend Integration

All endpoints return `application/json`. CORS is open for local development.

```typescript
// Generate a forecast for July–August 2026 on A8 East outbound
const res = await fetch("http://localhost:8000/api/forecast", {
  method: "POST",
  headers: { "Content-Type": "application/json" },
  body: JSON.stringify({
    corridor: "A8E",
    direction: "outbound",
    date_from: "2026-07-01",
    date_to: "2026-08-31",
  }),
});
const { success, forecast } = await res.json();

// Get top 10 critical peak days
const peakRes = await fetch(
  "http://localhost:8000/api/peak-days?corridor=A8E&direction=outbound&top_n=10"
);
const { peak_days } = await peakRes.json();
```

## Error Responses

| HTTP Code | When |
|-----------|------|
| 404 | Dataset file not found in `data/raw/` |
| 422 | Invalid request parameters (e.g. unknown corridor, date out of range) |
| 500 | Model not loaded or internal processing error |
