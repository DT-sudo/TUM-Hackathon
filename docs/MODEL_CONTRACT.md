# Model Contract

This document is the interface contract between:
- `scripts/train_model.py` — produces the model
- `models/prediction_pipeline.joblib` — the serialized pipeline
- `backend/app/` — loads and uses the model
- `frontend/` — displays the prediction result

Fill this file together with `scripts/train_model.py` on hackathon day.

---

## Model File

```
models/prediction_pipeline.joblib
```

Generated by running `python scripts/train_model.py` **before** starting the application.  
The application loads this file at runtime — it does **not** train the model.

---

## What the Pipeline Includes

- [ ] Preprocessing (scaling, encoding)
- [ ] Missing-value handling
- [ ] Feature transformation
- [ ] Trained estimator

---

## Expected Input

| Field | Type | Required | Accepted Values | Description |
|-------|------|----------|-----------------|-------------|
| [field_1] | float | Yes | any | [description] |
| [field_2] | string | Yes | ["a", "b", "c"] | [description] |

### Example Input

```json
{
  "field_1": 0.0,
  "field_2": "example_value"
}
```

---

## Loading the Model

```python
import joblib
import pandas as pd
from pathlib import Path

model = joblib.load(Path("models/prediction_pipeline.joblib"))
```

## Running a Prediction

```python
input_df = pd.DataFrame([{
    "field_1": 0.0,
    "field_2": "example_value"
}])

prediction = model.predict(input_df)
probabilities = model.predict_proba(input_df)  # if classifier
```

---

## Expected Output

```json
{
  "prediction": "class_label",
  "confidence": 0.87
}
```

| Field | Type | Description |
|-------|------|-------------|
| prediction | string / float | Primary prediction value |
| confidence | float 0–1 | Model confidence (if classifier) |

---

## Error Cases

| Error | Cause | Fix |
|-------|-------|-----|
| `FileNotFoundError` | .joblib not found | Run `python scripts/train_model.py` first |
| `ValueError: could not convert` | Wrong input type | Check field types against this contract |
| `KeyError` | Missing feature column | All required fields must be present |
